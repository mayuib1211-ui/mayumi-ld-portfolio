保護ねこサイト（自主制作）

direction / Design / Coding ｜ 2026.02 ｜ Figma / VS Code

タイプ
Webサイト
概要
保護猫の里親を募集するためのWebサイトです。猫の一覧閲覧・里親申し込みフォームを実装し、ユーザーが直感的に操作できるよう情報設計とUI設計を行いました。温かみのあるカラーパレットを採用し、サイト全体に統一感を持たせています。
制作期間
2026.02 ｜ 約3週間
対応デバイス
PC / タブレット / スマートフォン
役割
direction / Design / Coding
使用ツール
Figma / VS Code
ターゲット
保護猫に関心のある20〜40代の個人・ファミリー層
目的
保護猫の認知拡大と里親希望者の増加。ユーザーが猫の情報を閲覧し、里親申し込みへとスムーズに進めるWebページとして機能することを目指しました。
デザイン意図
デザインの軸は「温かみと信頼感」。全ページ共通でベージュ・アースカラーを基調とし、角丸や余白を多めにとることで柔らかい印象を演出しました。PC・スマートフォンどちらでも見やすいレイアウトに仕上げています。
工夫した点
猫一覧ページではカード型レイアウトを採用し、スクロールしながら複数の猫を比較しやすくしました。申し込みフォームにはバリデーションを実装し、入力ミスを防ぐ設計にしています。
