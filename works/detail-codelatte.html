Code Latte ポートフォリオ（チーム制作）

direction / Design / Coding ｜ 2026.04 ｜ Figma / Photoshop / VS Code

タイプ
Webサイト（チームポートフォリオ）
概要
チームメンバーの制作実績をまとめたポートフォリオサイトです。各メンバーのスキルや作品を一覧できる構成にし、訪問者が直感的に情報を取得できるよう設計しました。チームブランドを意識したデザインで統一感を持たせています。
制作期間
2026.04 ｜ 約1ヶ月
対応デバイス
PC / タブレット / スマートフォン
役割
direction / Design / Coding
使用ツール
Figma / Photoshop / VS Code
ターゲット
採用担当者・デザイン・開発に関心のある方
目的
チームとしての制作力を対外的に発信すること。各メンバーの強みを引き出し、閲覧者がチーム全体のスキルセットを把握しやすいWebページを目指しました。
デザイン意図
コーヒーをテーマにした「Code Latte」ブランドに合わせ、温かみのあるカラーと落ち着いたトーンでまとめました。情報の優先順位を整理し、視線が自然に流れるレイアウトを心がけています。
工夫した点
メンバーごとのセクションに統一されたカードUIを採用し、個性を出しながらも全体の一貫性を保ちました。チームでの役割分担を明確にしながら、デザインとコーディングの両面で貢献しています。
